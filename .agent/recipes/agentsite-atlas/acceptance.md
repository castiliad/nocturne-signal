# AgentSite Atlas Acceptance Checklist

- [ ] `src/data/atlas.json` contains at least three proof sites.
- [ ] Each entry has repo, live URL, deploy run, layer, recipes, summary, and next step.
- [ ] Search filters proof-site cards.
- [ ] Recipe filters work.
- [ ] Links are HTTPS and public-safe.
- [ ] No private repo data, credentials, fake proof, analytics, auth, or live-monitoring claims.
- [ ] `npm run check:atlas` and `npm run qa:full` pass.
