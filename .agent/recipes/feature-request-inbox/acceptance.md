# Feature Request Inbox Acceptance Checklist

- [ ] `src/data/requests.json` contains at least three requests.
- [ ] Each request records status, priority, source, summary, acceptance criteria, recipe candidates, verification plan, and next action.
- [ ] Search filters request cards.
- [ ] Status and priority filters work.
- [ ] No private tickets, raw chats, credentials, private data, analytics, auth, DB, or live-sync claims.
- [ ] `npm run check:requests` and `npm run qa:full` pass.
