# Roadmap Board Acceptance Checklist

- [ ] `src/data/roadmap.json` exists with at least three public-safe items.
- [ ] Every item has title, status, priority, owner, tags, summary, and nextStep.
- [ ] Search filters roadmap cards.
- [ ] Priority buttons filter cards.
- [ ] Local status select moves a card between columns.
- [ ] Reset local changes control is visible.
- [ ] Copy clearly says browser changes are local-only/non-authoritative.
- [ ] No fake automation, private data, analytics, auth, payments, or unsupported guarantees.
- [ ] `npm run check:roadmap` and `npm run qa:full` pass.
