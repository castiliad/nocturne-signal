# Copy Evidence Strip

A composable AgentSite recipe for making static-site copy feel more trustworthy without adding fake social proof.

## What it does

The recipe adds a horizontal proof band that pairs each visible allowed claim with a concrete artifact:

```text
Claim → Evidence artifact
```

Examples of acceptable evidence:

- repo contracts
- QA scripts and command output
- deploy verification notes
- screenshots generated by visual QA
- runbooks
- plans
- user-provided facts
- documented scope boundaries

## Triggers

Use explicitly:

```json
{
  "recipes": ["copy-evidence-strip"]
}
```

or:

```json
{
  "visualPreset": "evidence-strip"
}
```

It is designed to compose with `product-cockpit`:

```json
{
  "recipes": ["product-cockpit", "copy-evidence-strip"]
}
```

## Static safety rules

Do not use this recipe to imply:

- customer endorsements
- customer logos
- revenue, conversion, or usage metrics
- live telemetry
- analytics results
- payment success
- benchmark superiority

If a claim does not have real evidence, weaken the claim or remove it.

## Visual intent

The strip should feel like a trust ledger: compact, premium, scannable, and explicit about why each claim is allowed.

It should not become a noisy testimonial carousel or fake proof wall.
