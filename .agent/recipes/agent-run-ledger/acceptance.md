# Agent Run Ledger Acceptance Checklist

- [ ] `src/data/runs.json` contains at least three runs.
- [ ] Each run records status, layer, delegated work, checks, commit, deploy evidence, outcome, and next action.
- [ ] Search filters run cards.
- [ ] Status filters work.
- [ ] No raw private logs, credentials, private data, or live-monitoring claims.
- [ ] `npm run check:runs` and `npm run qa:full` pass.
